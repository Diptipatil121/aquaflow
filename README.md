# AquaFlow - Smart Water Distribution System

Production-ready frontend for Smart Water Distribution and Leakage Prevention.

## Tech Stack

- React 19 + TypeScript + Vite
- Tailwind CSS 4
- React Router 7
- TanStack Query
- Axios
- React Hook Form + Zod
- Framer Motion
- Recharts
- React Leaflet
- Sonner

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Features

- Dashboard with animated metrics and interactive map
- Leak detection with AI predictions
- Water distribution monitoring
- Reservoir level tracking
- Water quality gauges and trends
- Analytics with multiple chart types
- Consumer management with search/filter/pagination
- Reports generation (PDF/Excel)
- Maintenance scheduling
- Alerts management
- AI insights and recommendations
- Settings (profile, notifications, thresholds, theme)
- Dark mode support
- Fully responsive layout

## Project Structure

```
src/
├── api/           # Axios instance & mock data
├── components/    # Reusable UI components
├── context/       # Auth, Theme, Notification contexts
├── hooks/         # Custom hooks
├── pages/         # Feature pages
├── routes/        # Router configuration
├── services/      # API service layer
├── types/         # TypeScript types
└── utils/         # Utilities & constants
```
